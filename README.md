# Mohamad Zamini Website

Official personal website for Mohamad Zamini, hosted with GitHub Pages.

## Overview

This repository contains a single-page portfolio site that highlights:

- research and industry experience in machine learning
- selected projects across LLMs, multimodal AI, interpretability, and NLP
- technical skills, education, and contact information

## Stack

- HTML
- CSS
- Vanilla JavaScript
- [Typed.js](https://github.com/mattboldt/typed.js/) for the rotating hero text

## Local Preview

From the repository root, run:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Deployment

The site is intended for GitHub Pages deployment from this repository.
